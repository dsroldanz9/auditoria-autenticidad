🧭 CÓMO SE HIZO ESTA INVESTIGACIÓN
1. Semilla. Partimos de cuentas víctima (que reciben ataques) y rastreamos quién les responde/ataca.
2. Señales por cuenta. Antigüedad, % de respuestas, ratio seguidos/seguidores, handle aleatorio, avatar por defecto, contenido repetido.
3. Clasificación. "Alta señal" = ≥3 señales acumuladas. Es probabilístico, no un veredicto.
4. Coordinación. Muchas cuentas atacando a las MISMAS víctimas con el MISMO mensaje = comportamiento de bodega.
Metodología y código abiertos en GitHub. Cualquiera puede auditarla o reproducirla.
Las clasificaciones son estimaciones basadas en señales públicas. No afirman intención ni identidad. Datos descargables arriba.
